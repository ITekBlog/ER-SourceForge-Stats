== Description ==

ER SourceForge Stats allows you to insert SourceForge download statistics into your page using a simple shortcode.

== Installation ==

* Upload "er-sourceforge-stats" folder to the "/wp-content/plugins/" directory
* Activate the plugin through the 'Plugins' menu in WordPress
* View Settings > ER SourceForge Stats for a usage tutorial.
